# 정보보안기사 문제 풀이 웹 애플리케이션 설계서

**작성일**: 2026-02-07

## 1. 프로젝트 개요

### 1.1 목적
정보보안기사 자격증 시험을 준비하는 사용자를 위한 문제 풀이 및 학습 플랫폼 제공

### 1.2 주요 기능
- 과목별 이론 학습 (MDX 기반)
- 과목별 문제 풀이 (객관식)
- 오답 노트 관리 (localStorage)
- 다크 모드 지원
- 반응형 디자인

### 1.3 배포 URL
- **프로덕션**: https://security-engineer.anveloper.dev

## 2. 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | React 19, TypeScript |
| Build Tool | Vite |
| Styling | Tailwind CSS v4, @tailwindcss/typography |
| Content | MDX (@mdx-js/rollup, remark-gfm, rehype-highlight) |
| Diagrams | Mermaid.js |
| State Management | localStorage (클라이언트 저장) |
| Routing | React Router v7 |
| Deployment | GitHub Pages |

## 3. 정보보안기사 시험 구성

### 3.1 필기시험 (5과목)
1. **시스템 보안**: 운영체제, 클라이언트/서버 보안
2. **네트워크 보안**: 네트워크 개요, 프로토콜, 공격/방어
3. **어플리케이션 보안**: 웹, 데이터베이스, 전자상거래 보안
4. **정보보안 일반**: 암호학, 접근통제, 보안 모델
5. **정보보안 관리 및 법규**: 정보보호 관리체계, 관련 법규

### 3.2 실기시험
- 정보보안 실무 (필답형)

## 5. 핵심 데이터 구조

### 5.1 이론 콘텐츠 (Post)
```typescript
interface Post {
  id: string;
  title: string;
  description: string;
  chapter: string;
  order: number;
  component: React.ComponentType;
}
```

### 5.2 문제 세트 (QuestionSet)
```typescript
interface QuestionSet {
  subject: string;      // "01-system-security"
  subjectName: string;  // "시스템 보안"
  chapter: string;      // "01-terminal-systems"
  chapterName: string;  // "단말 시스템"
  questions: Question[];
}
```

### 5.3 문제 (Question)
```typescript
interface Question {
  id: number;
  question: string;
  options: string[];
  answer: number;       // 1-based index
  explanation: string;
  keywords: string[];
}
```

### 5.4 오답 기록 (WrongAnswer)
```typescript
interface WrongAnswer {
  questionSetId: string;  // "subject/chapter"
  questionId: number;
  selectedAnswer: number;
  timestamp: number;
}
```

## 6. 페이지 구성

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 홈 | `/` | 메인 대시보드, 기능 카드 |
| 이론 목록 | `/theory` | 과목별 이론 진입 |
| 이론 챕터 | `/theory/:subject` | 챕터별 이론 목록 |
| 이론 상세 | `/theory/:subject/:postId` | MDX 렌더링 |
| 문제 목록 | `/quiz` | 과목별 문제 진입 |
| 문제 챕터 | `/quiz/:subject` | 챕터별 문제 세트 |
| 문제 풀이 | `/quiz/:subject/:chapter` | 문제 풀이 UI |
| 오답 노트 | `/wrong-answers` | 틀린 문제 복습 |
| 모의고사 | `/mock-exam` | 실전 모의고사 |

## 7. 이론 콘텐츠 시스템

### 7.1 MDX 기반 콘텐츠
- 마크다운 + JSX 조합
- remark-gfm: GFM 테이블, 취소선 지원
- rehype-highlight: 코드 하이라이팅
- Mermaid: 다이어그램 지원
- Tailwind Typography: 스타일링

### 7.2 콘텐츠 구조
```
src/content/theory/{과목}/
├── index.ts          # Post 배열 export
├── 01-*.mdx          # 섹션별 MDX 파일
├── 02-*.mdx
└── ...
```

## 10. 배포 전략

- **develop 브랜치**: 개발 및 테스트
- **main 브랜치**: GitHub Actions를 통한 GitHub Pages 자동 배포
- **커스텀 도메인**: security-engineer.anveloper.dev

### 10.1 GitHub Actions 워크플로우
```yaml
# .github/workflows/deploy.yml
on:
  push:
    branches: [main]

jobs:
  deploy:
    - pnpm install
    - pnpm run build
    - Deploy to GitHub Pages
```

## 11. 반응형 디자인

### 11.1 지원 화면 크기
- 모바일: 320px ~
- 태블릿: 640px ~
- 데스크톱: 1024px ~

### 11.2 반응형 요소
- 헤더: 모바일 햄버거 메뉴
- 테이블: 가로 스크롤 + 줄바꿈
- 코드 블록: 가로 스크롤
- 폰트 크기: 모바일에서 축소
- 스크롤 버튼: 고정 위치
