### 기능 추가

#### 오답노트 페이지 (`/wrong-answers`)
- 과목별 그룹핑 표시
- 오답 카드: 문제 텍스트, 내 선택(빨간색) vs 정답(초록색)
- 해설 토글, 개별 삭제, 전체 삭제 (확인 단계)
- 다시 풀기 링크

#### 모의고사 페이지 (`/mock-exam`)
- 과목당 최대 20문제 랜덤 출제
- 3단계 흐름: 시작 화면 → 시험 진행 → 결과 화면
- 경과 시간 타이머
- 과목별 문제 네비게이터
- 미제출 문제 경고 모달
- 과목별 점수 + 합격/불합격 판정 (과목 40점, 평균 60점)
- 틀린 문제 해설 + 오답노트 자동 저장

#### 페이지별 SEO
- `usePageMeta` 훅: document.title + meta description 동적 설정
- `index.html`에 OG/Twitter 기본 메타태그 추가
- 전 9개 페이지에 페이지별 title/description 적용

#### 정적 페이지 프리렌더링
- `scripts/generate-static-pages.mjs` 빌드 후 스크립트
- 이론/문제 콘텐츠 자동 탐색하여 라우트별 HTML 생성
- title, description, OG, Twitter 메타태그 교체
- 소셜 크롤러(카카오톡, Facebook 등)에서 페이지별 메타 정보 노출
