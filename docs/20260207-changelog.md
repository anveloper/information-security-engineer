# 변경 로그

## 2026-02-07

### 기능 추가

#### 문제 풀이 시스템
- 객관식 문제 풀이 UI 구현
- 정답 확인 및 해설 표시 (키워드 태그 포함)
- 진행률 표시 및 문제 번호 네비게이터
- 오답 자동 저장 (localStorage)
- 1과목 문제 데이터 추가 (단말시스템, 서버시스템, 운영체제)

#### 다크 모드
- 시스템 테마 감지 (초기값)
- 수동 토글 버튼 (해/달 아이콘)
- localStorage에 설정 저장
- 전체 페이지 다크모드 스타일 적용

#### UI/UX 개선
- 공통 레이아웃 및 헤더 컴포넌트
- 반응형 네비게이션 (모바일 햄버거 메뉴)
- 헤더에 로고 아이콘 추가
- 위/아래 스크롤 버튼 (이론/문제 페이지)
- 페이지 이동 시 스크롤 상단 초기화

#### 콘텐츠 렌더링
- 코드 하이라이팅 (rehype-highlight)
- Mermaid 다이어그램 지원
- 인라인 코드 스타일 개선
- 테이블 반응형 (스크롤 + 줄바꿈)

### 구조 변경

#### 라우팅 분리
- 이론: `/theory` → `/theory/:subject` → `/theory/:subject/:postId`
- 문제: `/quiz` → `/quiz/:subject` → `/quiz/:subject/:chapter`
- 기존 `/subjects` 구조 제거

#### 디렉토리 구조
- `src/hooks/` 추가 (use-dark-mode)
- `src/content/questions/` 추가 (JSON 기반 문제 데이터)
- 절대 경로(@/) 설정

### 이론 콘텐츠
- 3과목 어플리케이션 보안 추가
- 4과목 정보보안 일반 추가
- 5과목 정보보안 관리 및 법규 추가

### 개발 환경
- 절대 경로(@/) 설정 (tsconfig, vite.config)
- ESLint 깊은 상대경로 import 경고 규칙
